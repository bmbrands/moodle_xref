   1  This files describes API changes in the lesson code.
   2  
   3  === 2.9 ===
   4  * The  grade_item_delete function in mod/lesson/lib.php was removed because it was not doing anything and was never called.
   5  * A third optional boolean parameter $endreached was added to lesson::update_timer to indicate that end of lesson was reached. This is used by 'completionendreached' custom completion rule.
   6  * lesson_page_type_endofbranch::update in mod/lesson/pagetypes/endofbranch.php
   7    and lesson_page_type_endofcluster::update in mod/lesson/pagetypes/endofcluster.php
   8    have been removed, this is now handled by the base class: lesson_page::update in
   9    locallib.php and the exact same code is executed (it is also executed by the
  10    lesson_page_type_cluster class that previously had no update function).
  11  * A fourth parameter (format) has been added to the add_answer() function
  12    located as part of the lesson_add_page_form_base class. If specified with a value of 'LESSON_ANSWER_HTML'
  13    then a rich html editor is generated. Otherwise an editor is created with Moodle Auto Format
  14  * removedoublecr() and importmodifiedaikenstyle() are now deprecated.
  15  * Added support for groups and groupings. If enabled, this allows the Report and
  16    Grade Essay view to be filtered by a group
  17
