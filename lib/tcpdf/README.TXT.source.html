  26  Description:
  27      TCPDF is a PHP class for generating PDF files on-the-fly without requiring external extensions.
  28      This library includes also a class to extract data from existing PDF documents and
  29      classes to generate 1D and 2D barcodes in various formats.
  30  
  31  Main Features:
  32      * no external libraries are required for the basic functions;
  33      * all standard page formats, custom page formats, custom margins and units of measure;
  34      * UTF-8 Unicode and Right-To-Left languages;
  35      * TrueTypeUnicode, OpenTypeUnicode v1, TrueType, OpenType v1, Type1 and CID-0 fonts;
  36      * font subsetting;
  37      * methods to publish some XHTML + CSS code, Javascript and Forms;
  38      * images, graphic (geometric figures) and transformation methods;
  39      * supports JPEG, PNG and SVG images natively, all images supported by GD (GD, GD2, GD2PART, GIF, JPEG, PNG, BMP, XBM, XPM) and all images supported via ImagMagick (http://www.imagemagick.org/script/formats.php)
  40      * 1D and 2D barcodes: CODE 39, ANSI MH10.8M-1983, USD-3, 3 of 9, CODE 93, USS-93, Standard 2 of 5, Interleaved 2 of 5, CODE 128 A/B/C, 2 and 5 Digits UPC-Based Extension, EAN 8, EAN 13, UPC-A, UPC-E, MSI, POSTNET, PLANET, RMS4CC (Royal Mail 4-state Customer Code), CBC (Customer Bar Code), KIX (Klant index - Customer index), Intelligent Mail Barcode, Onecode, USPS-B-3200, CODABAR, CODE 11, PHARMACODE, PHARMACODE TWO-TRACKS, Datamatrix, QR-Code, PDF417;
  41      * JPEG and PNG ICC profiles, Grayscale, RGB, CMYK, Spot Colors and Transparencies;
  42      * automatic page header and footer management;
  43      * document encryption up to 256 bit and digital signature certifications;
  44      * transactions to UNDO commands;
  45      * PDF annotations, including links, text and file attachments;
  46      * text rendering modes (fill, stroke and clipping);
  47      * multiple columns mode;
  48      * no-write page regions;
  49      * bookmarks, named destinations and table of content;
  50      * text hyphenation;
  51      * text stretching and spacing (tracking);
  52      * automatic page break, line break and text alignments including justification;
  53      * automatic page numbering and page groups;
  54      * move and delete pages;
  55      * page compression (requires php-zlib extension);
  56      * XOBject Templates;
  57      * Layers and object visibility.
  58      * PDF/A-1b support.
  59  
  60  Installation (full instructions on http:  www.tcpdf.org):
  61      1. copy the folder on your Web server
  62      2. set your installation path and other parameters on the config/tcpdf_config.php
  63      3. call the examples/example_001.php page with your browser to see an example

  90  Third party fonts:

  92      This library may include third party font files released with different licenses.
  93      
  94      All the PHP files on the fonts directory are subject to the general TCPDF license (GNU-LGPLv3),
  95      they do not contain any binary data but just a description of the general properties of a particular font.
  96      These files can be also generated on the fly using the font utilities and TCPDF methods.
  97      
  98      All the original binary TTF font files have been renamed for compatibility with TCPDF and compressed using the gzcompress PHP function that uses the ZLIB data format (.z files).
  99      
 100      The binary files (.z) that begins with the prefix "free" have been extracted from the GNU FreeFont collection (GNU-GPLv3).
