   3  Compiling mimeTeX:
   4  
   5  Windows
   6  =======
   7  1/ get "Automated MinGW Installer" from https://sourceforge.net/projects/mingw/files/
   8  2/ install mingw
   9  3/ go into directory with extracted source files
  10  4/ execute "set path=%path%;c:\mingw\bin"
  11  5/ execute "c:\mingw\bin\gcc -DAA -DWINDOWS mimetex.c gifsave.c -lm -o mimetex.exe"
  12  
  13  Linux
  14  =====
  15  1/ install gcc
  16  2/ go into directory with extracted source files
  17  3/ execute "cc -DAA mimetex.c gifsave.c -lm -o mimetex.linux"
  18  
  19  FreeBSD
  20  =======
  21  1/ go into directory with extracted source files
  22  2/ execute "cc -DAA mimetex.c gifsave.c -lm -o mimetex.freebsd"
  23  
  24  Apple OSX
  25  =========
  26  1/ install XCode and command line tools
  27  2/ go into directory with extracted source files
  28  3/ execute "cc -DAA -arch i386 -arch x86_64 -mmacosx-version-min=10.5 mimetex.c gifsave.c -lm -o mimetex.darwin"
